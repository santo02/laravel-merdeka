## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

- Clone project dengan menjalankan perintah `git clone https://gitlab.com/santolamsar/laravel-merdeka.git` di terminal/cmd
- creat new database di localhost dan import database `laravel-merdeka` ke localhost
- Pastikan sudah menginstall XAMPP atau bisa download pada link berikut `https://www.apachefriends.org/download.html`
- Setelah proses clone berhasil, buka folder project di directory anda
- Buka terminal dan arahkan ke folder project
- Jalankan project dengan menjalankan perintah : `php artisan serve`
- Buka http://127.0.0.1:8000/ di browser
- Project berhasil dijalankan
